## Pure functions:
---
### Characterstics
1. Only access local variables 
2. No side effects (mutate or change varaible outside of the scope)
3. Given same input will always return the same output how manyevery times you call

### Benifits
1. Reliability
2. Simplicity

## Immutability
---
Allows to do
- State snapshots
- Undo changes
- Reload from URL
- Time travel
